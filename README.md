## Getting Started

## Prerequisites

  ```sh
  **Node.js**: Scarica e installa Node.js da [nodejs.org](https://nodejs.org/). Verifica l'installazione con:
    node -v
    npm -v
    git -v
  ```

## Installation

```sh
  git clone https://github.com/Simobara/F_ITA_SerieA.git
  cd ITA_SerieA
  npm install
  npm run dev
```
